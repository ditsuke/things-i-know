#cli/linux #coreutils

Check disk usage by files/directories:

```man
NAME
       du - estimate file space usage

SYNOPSIS
       du [OPTION]... [FILE]...
       du [OPTION]... --files0-from=F

DESCRIPTION
       Summarize device usage of the set of FILEs, recursively for directories.
```

Look at [[dust]] for a friendlier alternative.
